Detailed Description

template<std::size_t ValueCount>
class friz::Animation< ValueCount >

This class owns a number of AnimatedValue objects. On each animation frame it gets the next calculated value from each of the value objects and passes those values to its OnUpdate handler. When all of the values in the animation have reached their end states, calls the OnCompletion handler.

Once this animation is complete, the Animator object that owns it will garbage collect it.
